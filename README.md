# 佳乐的记录站

这是一个给佳乐自己、老师、同学、朋友以及其他想了解佳乐的人看的长期个人网站。

它不是课程作业页，也不是一次性简历。网站围绕一条主线展开：

> 我是谁 -> 我相信什么 -> 我怎样成长 -> 我沉淀了哪些方法 -> 我怎样生活

## 页面结构

| 页面 | 文件 | 职责 |
| --- | --- | --- |
| 首页 | `index.html` | 建立第一印象，呈现核心座右铭并把访客引向其他页面 |
| 关于我 | `about.html` | 长期更新自我介绍、关注方向和座右铭解释 |
| 成长记录 | `efforts.html` | 记录真实行动中的阶段、收获和变化 |
| 方法记录 | `results.html` | 沉淀 AI 协作、研究写作和项目推进方法 |
| 生活碎片 | `fragments.html` | 保存短句、近况、照片和日常感受 |

所有页面使用：

- `css/styles.css`：全站共用样式
- `js/site.js`：全站导航、页脚和进入动画

## 内容放置规则

### 写进“成长记录”

适合放：

- 正在做的项目
- 当前进展
- 遇到的问题
- 下一步计划
- 项目过程中的反思

### 写进“方法记录”

适合放：

- AI 协作方法
- 学习与研究方法
- 项目推进方法
- 写作、复盘和判断方式

项目记录先写阶段和收获，适合公开的材料以后再慢慢补。

### 写进“生活碎片”

不用写成长文章，回答三件事就够了：

1. 什么时候发生？
2. 当时真实感受到什么？
3. 以后为什么还想记得？

## 推荐更新节奏

- 有真实进展时：更新一次“成长记录”
- 形成可复用方法时：整理进“方法记录”
- 想记下一刻时：增加一条“生活碎片”
- 每学期：重写一次“关于我”
